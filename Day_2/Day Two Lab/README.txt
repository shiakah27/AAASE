============================================================
PROJECT OVERVIEW
============================================================

This project implements a real enterprise-grade Autonomous
Research AI Agent using modern AI orchestration frameworks.

The system autonomously:
- Collects research information
- Searches the web
- Stores memory in vector databases
- Analyzes multiple sources
- Evaluates research quality
- Dynamically adapts workflows
- Generates enterprise reports

============================================================
SYSTEM ARCHITECTURE
============================================================

The AI workflow uses graph-based orchestration.

Workflow Steps:
----------------

1. Research Collection Node
   - Searches external sources autonomously

2. Memory Storage Node
   - Stores collected data into vector memory

3. Analysis Node
   - Uses LLMs to analyze information

4. Quality Evaluation Node
   - Evaluates research quality dynamically

5. Dynamic Routing
   - Decides whether to recollect data
   - Or continue report generation

6. Report Generation Node
   - Produces enterprise reports

7. Audit Node
   - Records execution logs

============================================================
INSTALLATION
============================================================

1. Install Python 3.10+

2. Install required packages:

pip install langchain
pip install langgraph
pip install langchain-openai
pip install langchain-community
pip install chromadb
pip install tavily-python
pip install python-dotenv

============================================================
API KEYS
============================================================

Create a .env file:

OPENAI_API_KEY=your_openai_api_key
TAVILY_API_KEY=your_tavily_api_key

HOW TO RUN
