SEQUENTIAL ENTERPRISE AI AGENT
============================================================

Author:
-------
Enterprise AI Workflow Example

Project Description:
--------------------
This project demonstrates a simple Sequential Enterprise
AI Agent system implemented in Python.

The system processes corporate tasks through a strict,
predefined workflow where every step must be completed
successfully before moving to the next step.

The workflow simulates a real enterprise approval and
execution pipeline.

============================================================
WORKFLOW ARCHITECTURE
============================================================

The workflow contains the following agents:

1. Validation Agent
   - Validates input data
   - Ensures required fields exist
   - Verifies request amount

2. Risk Assessment Agent
   - Evaluates business risk
   - Rejects high-risk requests

3. Compliance Agent
   - Applies corporate policy rules
   - Prevents restricted operations

4. Approval Agent
   - Performs management approval
   - Approves or rejects requests

5. Execution Agent
   - Executes approved tasks
   - Updates execution status

6. Audit Agent
   - Records audit information
   - Maintains enterprise logs
